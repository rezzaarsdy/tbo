Chomsky Hierarchy

Chomsky, Noam Chomsky melakukan penggolongan tingkatan dalam bahasa berdasarkan aturan produksi,yaitu menjadi empat class yang disebut Hirarki Chomsky. Penggolongan tingkatan yang dilakukan oleh Noam Chomsky ada 4 class yaitu:

  • Tipe 0 / Unrestricted : tidak ada batasan pada aturan produksi
  • Tipe 1 / Context sensitive : panjang string ruas kiri harus < (lebih kecil) atau = (sama dengan) ruas kanan
  • Tipe 2 / Context Free Grammar : ruas kiri haruslah tepat satu symbol variabel, yaitu simbol non terminal
  • Tipe 3 / Regular : ruas kanan hanya memiliki maksimal satu symbol non terminal dan diletakkan paling kanan sendiri

Tipe 0 (Unrestricted) , Disebut bahasa formal, dan dikenal oleh mesin turing Memiliki aturan dimana simbol ruas sebelah kiri harus minimal ada sebuah simbol variabel dan tidak ada batasan pada aturan produksi.
Example :
Abc -> Dc
ABC -> b
abc -> GH



Tipe 1 (Context Sensitive),digunakan untuk mewakili bahasa peka konteks. Memiliki aturan dimana Simbol sebelah kiri harus simbol variabel.
Example :
Ab -> Def
ABC -> de

Tipe 2 (Context Free Grammar),Bahasa yang dihasilkan oleh tata bahasa dikenali oleh automata Pushdown. Dan menghasilkan bahasa tanpa konteks.
